This module and Admin Tab allows the display of (random) linked adverts / creatives in the side bar

Another great module from www.ecartservice.net!

To use, please follow these steps:

  1. Upload all the files from the archive to your site's shop root folder
  2. In the Back Office go to Tools-->Tabs, click "Add New" and enter the details as in Figure 1 (below)
  3. Again, in the Back Office, go to the Modules tab and click install for the "Adverts" module.
  4. (optional) Configure the adverts module to enable random display
  5. Go to the Adverts tab and insert your linked adverts (example images are contained in the adverts module directory and show in Figs. 2 and 3 below)
  6. Start using your adverts on your Site!!
